# Smart-City-Risk-Analysis-System
Σύστημα συλλογής και ανάλυσης δεδομένων πόλης σε πραγματικό χρόνο που συνδυάζει μετεωρολογικά δεδομένα, ποιότητα αέρα και κυκλοφορία. Το σύστημα εφαρμόζει αλγόριθμο K-Means clustering για την κατηγοριοποίηση επιπέδων κινδύνου και αποστέλλει ειδοποιήσεις μέσω MQTT και RabbitMQ. Υλοποιήθηκε σε Node-RED με ενσωμάτωση πολλαπλών APIs και pipelines.
